# SimpleCalculator

This Java project is a simple calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division based on user input. It demonstrates how to take input from the user, use conditional statements, and execute different operations using a simple menu system.

## Key Features:
- Takes two numbers as input
- Allows the user to choose an operation (addition, subtraction, multiplication, or division)
- Performs the chosen operation and displays the result
- Handles invalid operations gracefully

## Usage:
Run the main class and input two numbers. Choose an arithmetic operation from the menu to perform the calculation. The program will display the result based on the chosen operation.

## Technologies Used:
- Java
